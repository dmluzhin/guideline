Menus

Menu types

Navbar Menu

Navbar-menu - тип меню, зафиксированный слева. Способен скрываться по нажатию на Burger (см. Icons). Многофункциональное меню, способное содержать в себе подменю. Так же относится к категории Layout, где будут подробнее описаны части ее верстки.

Navbar-menu способен содержать в себе несколько разделов меню. Так в SHELL2.0 Navbar-menu отображает тройную вложенность:

Dropdown menu

Dropdown menu - используется для различных действий в CMS. Определяется Dropdown menu с помощью иконки DOTS. По клику на DOTS выпадает меню с различными пунктами. Основными пунктами являются Add, Edit и Delete.

Все пункты Dropdown menu обозначаются иконками слева и текстом справа.

Switcher

Switsher (более известный, как Tabs) позволяет безболезненно переключаться между понктами меню. Был реализован в ранней версии SHELL CMS, но пока признан ненужным.

Имеет аналог в пункте Buttons, который призван переключать тип модуля (см. Radiobutton-checkbox).
