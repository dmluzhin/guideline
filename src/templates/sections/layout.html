Text Fields

Text Field - редактируемое текстовое поле. В код встраивается, как обычный блок с атрибутом contenteditable, что позволяет превратить данный блок в текстовое поле, заполняемое с клавиатуры.

В Text Field есть возможность вставлять практические любые текстовые элементы. Добавляются они с помощью элемента управления Tools.

    Типы контента, допустимые в Text Field:
  • Simple Text - Обычный текст, обернутый в тэг div
  • Bold Text - Обычный текст, обернутый в тэг strong
  • Italic Text - Обычный текст, обернутый в тэг em
  • Undeline Text - Текст с нижним подчеркиванием. Обычный текст оборачивается в тэг u
  • Bullet List - Список с буллитами
  • Numbered List - Нумерованный список
  • Table - Таблица
  • Popup - Встраевомый попап (высплывающее окно)
  • Lightbox
