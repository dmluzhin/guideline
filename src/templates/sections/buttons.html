Buttons

Button types

Ниже указаны виды кнопок, используемые в SHELL 2.0 и CMS. Для удобства показаны все возможные состояния (standart, hover, active, disabled) и цветовые решения в зависимости от выбранной цветовой схемы, а так же универсальные решения.

Для удобства рядом с кнопкой реализован SCSS код.

Simple button

Обычные кнопки, которые можно использовать для чего угодно. Никаких иконок, только текст и анимация.

Switcher

Кнопки типа Switcher необходимы для быстрого выбора типа модуля.

  • SCSS
Edit on codepen 
      
				.selector {
					display: flex;
					justify-content: center;
					align-items: center;
					input[type=radio] {
					display: none;
					}
					label {
					display: inline-block;
					}
					input[type=radio] + label {
					position: relative;
					width: 150px;
					height: 40px;
					display: flex;
					justify-content: center;
					align-items: center;
					font-size: 14px;
					text-transform: uppercase;
					font-weight: 500;
					color: #212121;
					text-align: center;
					cursor: pointer;
					transition: all .25s;
					border: 2px solid #9d1f60;
					&:active {
					background-color: #9d1f60;
					color: white;
					}
					}
					label[for="NAME-1"] {
					border-radius: 2px 0 0 2px;
					}
					label[for="NAME-2"] {
					border-radius: 0 2px 2px 0;
					}
					input[type="radio"]:checked + label {
					background-color: #9d1f60;
					color: white;
					}
				}
